#  Personal Portfolio Website

A personal portfolio website built with **Python** and **Django**.

The project was created to practice Django fundamentals, including URL routing, templates, views, and website structure.

##  Features

- Responsive dark-themed design
- Single-page portfolio layout
- Navigation menu with smooth scrolling
- About Me section
- Skills section
- Projects section
- Contact information
- Built using Django Templates

##  Technologies

- Python 3
- Django
- HTML5
- CSS3
- Bootstrap 5

## Installation

Clone the repository:

```bash
git clone https://github.com/MiraMadani/DjangoResume.git
```

Go to the project directory:

```bash
cd DjangoResume
```

Install Django:

```bash
pip install django
```

Run migrations:

```bash
python manage.py migrate
```

Start the development server:

```bash
python manage.py runserver
```

Open in your browser:

```
http://127.0.0.1:8000/
```
